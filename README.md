# 🎬 AI Video Frames Effect Website

An interactive **video frame effect website** built using **HTML, CSS, and JavaScript**, with visual assets and concepts generated using **Google Labs (Whisk & Flow)**.

This project creates a cinematic **video-style frame transition effect** for websites, turning static visuals into immersive storytelling experiences.

## ⚙️ How It Works

1. **Google Labs (Whisk + Flow)** generates visual storytelling scenes.
2. Frames are exported and integrated into the website.
3. **JavaScript controls animation timing and effects.**
4. **CSS handles glass UI and transitions.**
5. HTML structures the layout.
